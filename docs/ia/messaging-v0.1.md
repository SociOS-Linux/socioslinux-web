# Messaging — v0.1

The canonical copy playbook for Socios Linux. Keep these consistent across every page.

## Primary message

> **Immutable host. Customizable user environment. Isolated agent plane.**

Secondary:

> Secure, local-first, agentic Linux stack built from the ground up.

## The three planes (30-second explainer)

A newcomer should be able to repeat this after 30 seconds on the homepage:

- **Host plane** — immutable OSTree-based core; atomic update/rollback; minimal trusted base.
- **User plane** — customizable workspace; local-first files; project environments; owner control.
- **Agent plane** — isolated from user space; capability-bounded; policy-governed; *not* ambient desktop privilege.

## Copy pillars (recur across the site)

- Owner control outranks convenience defaults.
- Local-first is the default, not offline-as-an-afterthought.
- Agents are separate from user space.
- The trusted base is small and recoverable.
- The system is customizable without dissolving the host boundary.
- Commons, org, and government lanes are distinct targets — not one page with different adjectives.

## Per-audience framing

- Dense enterprise procurement language.
- Exhaustive agent-framework internals.
