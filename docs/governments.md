# Governments

Governments are a separate target surface because their trust, sovereignty, and operational requirements are distinct.

## This page should eventually cover

- sovereignty posture
- local-first and disconnected operation
- explicit trust boundaries
- institutional durability
- public-sector deployment fit

## Why it is separate

A public-interest system and a government-capable system can overlap, but they should not be narrated as the same buyer, the same control plane, or the same accountability environment.
