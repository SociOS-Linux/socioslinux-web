# FAQ

## Is this just a Linux distribution?

No.

The public posture is a Linux stack with a distinct architectural model for host, user, and agent execution.

## Why separate agent space from user space?

Because agentic execution has different risk, control, and observability requirements than ordinary applications.

## How does this relate to the broader stack?

It is part of a connected system, but this site is intentionally scoped to the Linux lane so the message stays legible.

## Why not explain everything on the homepage?

Because the first job of the homepage is orientation, not total narrative compression.
